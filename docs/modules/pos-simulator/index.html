POS Simulator Feature (SAIAPP3-56)

Comprehensive Point of Sale simulation module enabling users to model special banking products, calculate online payment equivalents, and explore POS payment options.

Overview

The POS Simulator feature allows customers and advisors to:

  • Select and simulate special banking products with configurable parameters
  • Calculate monthly payments, total cost, and interest breakdown
  • View equivalent amounts for online payment channels
  • Explore POS payment options with installment plans
  • Compare payment methods before committing

Technical Foundation

  • Frontend: React 18 SPA page with form validation and results display
  • Backend: ASP.NET Core controller with POST /api/pos-simulator/calculate
  • Routing: React Router at /pos-simulator
  • i18n: Spanish, English, Portuguese support
  • Styling: Tailwind CSS with responsive design
