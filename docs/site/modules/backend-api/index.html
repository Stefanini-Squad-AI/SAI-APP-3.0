backend-api

ASP.NET Core Web API responsible for authentication, business operations, and admin domain services used by frontend clients.

Technical foundation

  • .NET 8 with ASP.NET Core controllers.
  • JWT authentication and authorization middleware.
  • CORS configured for localhost, GitHub Pages, and Surge previews.
  • Swagger enabled in development for API exploration.

Dependencies and integration points

  • MongoDB repositories and infrastructure service registration through extension methods.
  • Default admin seeding on startup (idempotent) for local environments.
  • Consumed by frontend admin/public services and CI smoke/functional flows.
