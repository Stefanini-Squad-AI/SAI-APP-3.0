frontend-admin

JWT-protected administration interface for operational workflows and catalog management.

Technical foundation

  • React + React Router protected nested routes under /admin.
  • Guarding via ProtectedRoute and context-based authentication.
  • Layout and navigation managed in AdminLayout.

Dependencies and integration points

  • Consumes backend APIs through service clients in frontend/src/services.
  • Relies on JWT issuance from backend auth endpoints.
  • Includes admin forms, modal workflows, and stateful list/detail operations.
