frontend-public

Public React experience used by end-users to explore services, calculate credits, submit contact requests, and read legal information pages.

Technical foundation

  • Framework: React 18 with Vite.
  • Routing: react-router-dom with routes defined in frontend/src/App.jsx.
  • Internationalization: i18next with en, es, and pt locales.
  • Styling: Tailwind CSS utility classes in shared layout components.

Dependencies and integration points

  • Uses API services in frontend/src/services for data and submissions.
  • Legal links and navigation entries are rendered in shared header/footer layouts.
  • Language selection persists with localStorage/cookie key tco-user-language.
  • Runtime language switching is provided by the header LanguageSelector component.
  • Supported runtime locales: en, es, and pt.
