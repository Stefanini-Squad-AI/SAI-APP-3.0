Módulo: Sitio Público (sitio-publico)

Propósito: Páginas públicas del sitio TuCreditoOnline: inicio, servicios, nosotros, FAQ, contacto, calculadora, y páginas estáticas legales (Privacidad, Términos, Información legal).

Base técnica

  • React 18, React Router, Vite.
  • Layout: MainLayout (Header + Outlet + Footer).
  • i18n: react-i18next con locales es, en, pt.

Navegación y flujos afectados

  • Header (desktop y móvil): Enlace “Información legal” (header.legalInfo) situado inmediatamente después de “Contacto”, apuntando a /legal.
  • Footer — sección Legal: “Aviso Legal” (footer.legalNotice) enlaza a /legal (no a /terms).

Dependencias

  • react-router-dom (Link, Route, useNavigate).
  • react-i18next (useTranslation, claves legalPage.*, header.legalInfo).
  • Sin llamadas a backend para la página Legal; contenido estático vía i18n.

Patrones de reuso

La página Información legal sigue el mismo patrón que TermsPage y PrivacyPage:

  • Hero con gradiente (from-primary-600 to-primary-800).
  • Contenedor centrado max-w-4xl, tarjeta con rounded-2xl shadow-soft.
  • Secciones con h2 y párrafo; al menos tres secciones de contenido.
  • Botones/enlaces finales: “Contáctanos” → /contact, “Volver al inicio” → / (reutilizando privacyPage.contactUs y notFoundPage.backHome).
