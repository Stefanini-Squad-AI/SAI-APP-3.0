System Overview

Architecture map, module catalog, routes, i18n boundaries, and integration surfaces.

Architecture and domain

  • frontend/: React 18 + Vite + Tailwind CSS single-page application.
  • backend/: ASP.NET Core Web API (.NET 8) with JWT authentication and MongoDB repositories.
  • database/: MongoDB initialization and seed support for local dockerized environments.
  • automation/: quality tooling, including applied-ai-aura functional automation.

Module catalog

Internationalization (strict)

  • Supported runtime locales are strictly: en, es, and pt.
  • Language switch is provided in the header selector and persisted with tco-user-language.
