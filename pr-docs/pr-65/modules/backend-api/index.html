backend-api

ASP.NET Core API that concentrates authentication, business operations, admin workflows, maintenance utilities, and health monitoring consumed by public and admin frontend modules.

Technical foundation

  • .NET 8 + ASP.NET Core controllers in backend/src/TuCreditoOnline.API/Controllers.
  • JWT authentication + role-based authorization (Admin, SuperAdmin).
  • Public and protected endpoints coexist for customer and admin experiences.
  • Controller methods delegate business logic to infrastructure/application services.

Controller domains and responsibilities

Controller Base route What it is responsible for
AuthController/api/authAdmin login and conditional user registration (first setup or admin-authorized creation).
DashboardController/api/dashboardKPI and distribution data for operations dashboard visualizations.
CreditRequestsController/api/creditrequestsCreation and lifecycle management of credit request records.
UsersController/api/usersAdmin user lifecycle: search, create, update, delete, and password updates.
CreditTypesController/api/credittypesCredit product catalog consumed by calculator and admin catalog tools.
ServicesController/api/servicesMarketing/service cards shown on public pages and managed by admins.
ContactMessagesController/api/contactmessagesCustomer contact intake and admin follow-up workflow by status.
BackupController/api/backupGenerate downloadable backups and report backup file inventory/status.
HealthController/api/healthAPI + database health telemetry for uptime checks.
